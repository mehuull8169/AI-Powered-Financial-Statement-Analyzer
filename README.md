# 💰 AI-Powered Financial Statement Analyzer

## 📌 Overview

The AI-Powered Financial Statement Analyzer is a machine learning-based system designed to analyze financial data, detect anomalies, and generate insightful reports. It uses AI models and data processing techniques to help users understand financial performance efficiently.

---

## 🚀 Features

* 📊 Financial data analysis and processing
* ⚠️ Anomaly detection using ML models
* 🤖 AI-generated financial insights
* 📈 Report generation system
* 🧠 Sentiment analysis on financial data
* 🌐 Interactive UI using Streamlit

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Frontend/UI:** Streamlit
* **Machine Learning:** Scikit-learn, PyTorch
* **Data Processing:** Pandas, NumPy
* **AI Integration:** LangChain (for intelligent responses)

---

## 📂 Project Structure

```
├── app.py                  # Main Streamlit app
├── backend.py              # Backend logic
├── anomaly.py              # Anomaly detection model
├── sentiment_analysis.py   # Sentiment analysis module
├── financial_tools.py      # Financial calculations
├── report_generator.py     # Report generation
├── langchain_integration.py# AI integration
├── prompts.py              # Prompt templates
├── financial_data.csv      # Sample dataset
├── scaler.pkl              # Pre-trained scaler
├── vae_financial.pth       # Trained ML model
├── requirements.txt        # Dependencies
├── run.py                  # Run script
```

---

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/mehuull8169/AI-Powered-Financial-Statement-Analyzer.git
cd AI-Powered-Financial-Statement-Analyzer
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the application

```bash
streamlit run app.py
```
